#  Porta lógica XOR usando os componentes: AND, NOT, e OR

XOR é a porta que dá 1 só quando A e B são diferentes. Com AND/OR/NOT

# Definição Lógica

### Tabela-verdade — Porta XOR

| A | B | Y  |
|:-:|:-:|:---------:|

| 0 | 1 |     1     |
| 1 | 0 |     1     |
| 1 | 1 |     0     |
